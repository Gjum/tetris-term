tetris-term
===========

Classic Tetris for your terminal.

Usage
-----

### Installation:

`git clone https://github.com/Gjum/tetris-term.git && cd tetris-term && make`

### Controls:

- `a`: move brick left
- `d`: move brick right
- `s`: rotate brick clockwise
- `w` or `Space`: drop brick down

Features
--------

- 7 different colors
- no fancy dependencies
- lightweight on your resources
